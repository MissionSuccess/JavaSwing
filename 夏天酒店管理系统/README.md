# 夏天酒店管理系统

##  1.项目说明
本项目为酒店管理系统，用于管理自家酒店，属JavaSwing项目。下载之后直接导入项目jdglxt至Eclipse即可

##  2.使用环境
Windows ALL、 Eclipse

##  3.数据库
### ① 数据库环境 
本项目使用Oracle数据库，具体配置查看项目下的jdbc.properties文件，可对其进行修改
### ② 数据库表
数据库表的建立可查看文件夹下的```夏天酒店管理系统-数据库表.xmind```文件

##  4.使用说明
导入项目之后打开org.lanqiao.jdmrg.control下的Test.java运行即可进入平台
